About

About this Extension

Essentials for KissAnime is an extension that allows the user to toggle off/on various elements of KissAnime as well as adding new options/features.

The extension was created on August 9th, 2014 (with no prior knowledge of html/css/javascript) with the only feature being removing the video page Ads with no option to disable it. Eventually with the help of a few friends and the internet, the ability to remove Ads and social media buttons throughout the entire site and ability to remove various video page elements was added. Two months later, the ability to toggle on/off these features was added as well as the Welcome and Pinned Box.

Fast-forward a few months and on May 18th, 2015 version 2.1 was made public after being private for almost 9 months. Most of the current features were added after the extension went public.

Custom Header Image

  • Image MUST be in .png, .gif, .tiff, or .bmp format.
  • Image MUST contain a Transparent Background.
  • Image WILL be cut off if the Slimmer Header option is enabled.
  • To prevent long loading times, it is suggested to NOT use a large image.
